# F-Casa META360TDAH

Formulário estático para famílias responderem pelo celular sobre rotina e comportamento familiar.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a raiz do repositório:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde o GitHub gerar o link.

O link final ficará parecido com:

```text
https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/
```

## Envio dos resultados

O formulário envia os dados para:

```text
https://formspree.io/f/xjgzpgoz
```

No primeiro envio, confira se o Formspree pediu confirmação do e-mail ou ativação do formulário.
